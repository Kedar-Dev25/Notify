## 🚨 Problem

Students waste time daily just to check their timetable:

Open WhatsApp → School Group → Media Tab → Scroll → Search timetable image → Confused → Repeat daily

❌ No automation  
❌ No reminders  
❌ High chance of missing class  

---

## 💡 Solution

Notify replaces manual checking with automated smart alerts.

✔ 10-minute advance notifications  
✔ Personalized updates based on branch & semester  
✔ No daily effort required  
✔ Reliable real-time delivery  

---

## ⚙️ Tech Stack

**Backend**
- Java 17
- Spring Boot
- Spring Scheduler (@Scheduled)
- Maven

**Frontend**
- React (Vite)
- Firebase Messaging

**Database**
- MySQL

**Cloud**
- Firebase Cloud Messaging (FCM)

---

## ✨ Features

- Smart push notification system  
- Branch & semester-based filtering  
- Automated scheduling system  
- Real-time Firebase alerts  
- Clean and scalable architecture
